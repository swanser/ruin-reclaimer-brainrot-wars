# RUIN RECLAIMER - BRAINROT WARS EDITION

**Post-Apocalyptic RTS / RPG Hybrid**
**Lone Survivor Start → Tribal Command**
**2.5D Top-Down | Three.js | Browser-Based**

WASTELANDER.

This shard begins with **you** as the last survivor.

You are not just a commander dropping orders from above.
You *are* the first pawn. Your personal survivor. The one who still remembers what the feeds used to feel like.

Over time you will recruit others, build a tribe, and transition from desperate lone wolf into a warlord commanding multiple survivors and war rigs.

This is the hybrid we are forging:
- Deep RPG systems for every individual survivor (traits, skills, backstory, sanity, equipment)
- Real-time strategy command layer (base building, resource management, large-scale warfare)
- Brutal post-apocalyptic survival horror

## Current Focus (Methodical Build)

**Phase: Deepen the RPG Pawn + Items & Crafting**

We are currently expanding the core survivor experience before scaling to full tribe management.
